# blazekart
BlazeKart – A modern Django-based e-commerce web application with authentication, cart, wishlist, checkout, and email verification.

BlazeKart is a full-featured Django e-commerce web application designed with a modern UI and secure backend architecture.

Features:
• Custom user authentication
• Email verification system
• Shopping cart & wishlist
• Order and checkout flow
• Category-based product filtering
• Admin dashboard
• Secure environment variable configuration
• Responsive UI design

Built with:
- Django
- Bootstrap
- SQLite
- SMTP email integration

This project follows secure development practices including hidden environment variables and production-ready settings configuration.
